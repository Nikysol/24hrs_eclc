# 24hrs EJE CENTRAL
Titulo 24hrs eje central
Descripción: 24 HRS EJE CENTRAL es un proyecto en el que registramos durante 24 horas el cruce peatonal del Eje Central para observar qué sucede a lo largo del día en términos de movilidad. Decidimos desarrollar una página web dividida en dos secciones.
